Prerequisites:
    GCC Compiler:
        13.3.0

To compile and run the project:
    make

To clean the project:
    make clean


Features being implemented:
    User Input:
        Depth of ORAM Tree
        Number of Concurrent Users

    Display:
        ORAMTree (Option 3)
        Stash (Option 4)
        PositionMap (Option 5)
        QueryLog (Option 6)
        DRLogSet (Option 7)


    Parallel Support:
        Asks the user number of threads that reads same block (Option )
